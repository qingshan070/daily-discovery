# 每日随机发现 (Daily Serendipity Engine)

随机发现你不知道自己想知道的信息——打破信息茧房的意外发现引擎。

## 触发条件

当用户提到以下任意关键词时触发：
- "每日发现" / "随机发现" / "帮我发现" / "来点新鲜的"
- "serendipity" / "discover something new" / "surprise me"
- "看看有什么好玩的" / "有什么我不知道的"
- "daily discovery"

## 执行流程

### Step 1: 随机选择方向

从以下 10 个大类 × 5 个子方向 = 50 个入口中，随机选择 10 个不同子方向（避免重复大类，确保覆盖面）：

| 大类 | 子方向 |
|------|--------|
| **科学与技术** | AI前沿、航天探索、生物技术、材料科学、能源革命 |
| **商业与创业** | 新商业模式、startup故事、投资逻辑、营销案例、组织管理 |
| **人文与哲学** | 伦理学、认知科学、东西方哲学、存在主义、意识研究 |
| **历史与考古** | 古代文明、世界史、人物传记、考古新发现、口述历史 |
| **艺术与设计** | 视觉艺术、工业设计、建筑设计、音乐创作、电影语言 |
| **社会与心理** | 社会现象、行为经济学、发展心理学、教育创新、城市研究 |
| **自然与地理** | 动物行为、植物世界、地质奇观、气候变迁、海洋探索 |
| **工程与制造** | 机械设计、电子工程、土木奇迹、航空技术、制造工艺 |
| **文化与生活** | 全球美食、旅行目的地、时尚变迁、运动竞技、游戏设计 |
| **未来与趋势** | 技术预测、科幻思想、人口趋势、城市未来、工作形态演变 |

### Step 2: 搜索内容

- **前 5 个子方向** → 用中文搜索词，优先中文平台（微信公众号、知乎、B站、小红书、36氪、虎嗅等）
- **后 5 个子方向** → 用英文搜索词，优先英文平台（Medium、YouTube、Reddit、Hacker News、The Verge、Nature、MIT Tech Review等）
- 每个子方向使用 WebSearch 搜索，搜索词格式：
  - 中文：`{子方向} 推荐 深度 2025 2026`
  - 英文：`best {sub-direction} article video 2025 2026 recommended`
- 从搜索结果中精选 1 条最有价值的内容
- 使用 WebFetch 获取目标页面标题和简介

### Step 3: 生成日報

生成 Markdown 文件，格式如下：

```markdown
# 🔍 每日随机发现 | YYYY-MM-DD

> 今天随机探索的 10 个方向，5 条中文 + 5 条英文，帮你打破信息茧房。

---

## 🇨🇳 中文发现

### 🎲 {子方向} | {大类}
**[{标题}]({链接})**
📄 文章 / 🎬 视频
> {一句话简介，概括核心看点}

### 🎲 {子方向} | {大类}
...

---

## 🇬🇧 English Discoveries

### 🎲 {Sub-direction} | {Category}
**[{Title}]({link})**
📄 Article / 🎬 Video
> {One-sentence summary}

### 🎲 {Sub-direction} | {Category}
...

---

*Generated by Daily Serendipity Engine · [View on GitHub](https://github.com/qingshan070/daily-discovery)*
```

### Step 4: 推送到 GitHub

将生成的 Markdown 文件保存并推送到 `qingshan070/daily-discovery` 仓库：
- 文件路径：`archive/{YYYY}/{YYYY-MM-DD}.md`
- 同时更新仓库顶部的 `LATEST.md` 指向最新一期

## 质量要求

1. **每个方向一条，不重复**
2. **优先选择近期热门/高质量内容**（近半年内）
3. **链接必须真实有效**，不可虚构
4. **简介一句话说清为什么值得看**，不要泛泛而谈
5. **文章和视频混合**，10 条中至少有 3 条视频内容
6. **来源多样化**，避免同一平台出现超过 2 次

## 搜索技巧

- 中文搜索可添加 `site:zhihu.com` 或 `site:mp.weixin.qq.com` 限定来源
- 英文可搜索 `site:medium.com` 或 `site:news.ycombinator.com`
- 视频内容搜索可加 `site:bilibili.com` 或 `site:youtube.com`
- 技术类可搜索 `site:github.com` 找 trending 项目
- 优先选择有深度的长文/长篇视频，避免碎片化内容
