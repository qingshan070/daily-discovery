# 🔍 每日随机发现 (Daily Serendipity Engine)

> *你无法搜索你不知道的东西。所以让随机性帮你。*

每天从 **50 个知识方向** 中随机抽取 10 个，从全球互联网精选 5 条中文 + 5 条英文最值得看的内容，帮你打破信息茧房。
